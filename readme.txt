== Description ==

A plugin contains of blocks to make your WordPress website more engaging. The blocks are :

1. Number counter (a block to count up the number displayed)
2. Running images (a block where you can add images and they will move infinitely)
3. Twinkling text (create a twinkling point beside the text)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/block-kuy` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
